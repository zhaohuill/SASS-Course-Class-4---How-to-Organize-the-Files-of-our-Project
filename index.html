How to Organize the Files of our Project:_

In Large Scale Webpage Projects wich have Thousands lines of Code, it is Highly Recommended to have the Modules of our Project well separated and organized, to be able to Modify them Individually in a much easier and organized way. In SASS this is achieved by Creating in Our Same Folder "sass" where our File "styles.scss" is. Files corresponding to each Module of the SASS Code whose names begin with an underscore "_" indicating that they are dependencies of the Main File, for example : _ "_sassVariables.scss", "_hader.scss", "_footer.scss", etc.
And then We connect them to the Main File "styles.scss" writing this at the beginning of the Code: _ @import "***"; Where *** represents the Section File Name Including the Quotation Marks and WITHOUT the Underscore "_" or the File Type Indication ".scss". For example:_
@import "sassVariables";
@import "header";
@import "footer";

You can Learn More by Reading the SASS Official Documentation.
